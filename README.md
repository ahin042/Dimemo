# 📑 디메모 (D-Memo)

[디메모 바로가기](https://ahin042.pythonanywhere.com/)

---

## 📌 프로젝트 소개

디메모(D-Memo)는 일기(Diary)와 메모(Memo) 기능을 결합한 개인 기록 공간입니다.  
단순히 글을 남기는 것을 넘어, 입력한 '사용자 이름'에 따라 각각 독립된 데이터를 불러오기 때문에 나만의 맞춤형 기록을 유지할 수 있습니다.

---

## ✨ 주요 기능

### 👤 사용자 기반 데이터 관리
* **사용자 변경:** 입력창에 이름을 입력하고 '사용자 변경하기'를 누르면, 해당 사용자가 작성했던 일기와 메모 데이터만 필터링하여 불러옵니다.

### ✍️ 일기 (Diary) 기능
* **날짜별 관리:** 날짜를 선택하여 하루의 이야기를 기록할 수 있습니다.
* **1일 1일기 & 수정:** 같은 날짜에 새로운 내용을 작성하면 기존 내용이 덮어쓰여(수정) **한 날짜에 오직 하나의 일기만** 깔끔하게 유지됩니다.
* **삭제 기능:** 작성한 일기를 간편하게 삭제할 수 있습니다.

### 📝 메모 (Memo) 기능
* **다중 작성:** 일기와 달리, **한 날짜에 여러 개의 메모를 동시에 추가**하고 관리할 수 있어 할 일 목록이나 아이디어 기록에 용이합니다.
* **리스트 확인:** 해당 날짜에 저장된 메모들을 아래에서 바로 확인할 수 있습니다.
